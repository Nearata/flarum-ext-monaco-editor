## Install

```sh
composer require nearata/flarum-ext-monaco-editor:"*"
```

## Update

```sh
composer update nearata/flarum-ext-monaco-editor:"*"
php flarum cache:clear
```

## Remove

```sh
composer remove nearata/flarum-ext-monaco-editor
php flarum cache:clear
```

## How to use

This adds Monaco Editor to:

- Custom Header modal
- Custom Footer modal
- Custom CSS modal
